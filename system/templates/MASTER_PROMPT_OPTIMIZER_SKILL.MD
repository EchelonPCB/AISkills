2. **Extract Intent**

   2.3 Identify constraints

3. **Structure**
   3.1 PURPOSE
   3.2 INPUTS
   3.3 OUTPUTS

   4.1 Remove ambiguity
   4.2 Add precision

5. **Output**
   5.1 Return structured prompt
   5.2 Ready for AI execution
